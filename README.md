# 🔗 DeliberateChain

### "Aspirasi yang Tak Bisa Dimanipulasi — Transparansi yang Terukir di Blockchain"

DeliberateChain adalah platform digital berbasis **AI + Blockchain** yang mengubah partisipasi publik dari sekadar pengaduan menjadi proses deliberasi demokrasi yang transparan, inklusif, dan aman dari manipulasi.

---

## 🌟 Highlights
- **AI-Powered Deliberation**: Ringkasan otomatis dan pengelompokan aspirasi menggunakan Gemini AI.
- **Blockchain Transparency**: Voting on-chain yang immutable (Sepolia Testnet).
- **Auto Policy Brief**: Ringkasan kebijakan instan untuk pemerintah.
- **Gasless Participation**: Masyarakat bisa berpartisipasi tanpa harus memiliki ETH.

---

## 🚀 Features

| Profile | Key Features |
|---------|--------------|
| **👤 Public** | Submit Aspirasi, Join Deliberation Room, Gasless Voting, Reward Civic Tokens. |
| **🏛️ Government** | Analitik Isu Hot, Responder Resmi, Dashboard Tindak Lanjut, Auto-generated Policy Brief. |
| **👑 Admin** | AI Prompt Tuning, Blockchain Management, Global Moderation. |

---

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **AI**: Google Gemini 2.5
- **Blockchain**: Ethereum (Sepolia) via Thirdweb SDK
- **Styling**: Tailwind CSS v4 (Premium Editorial Design)
- **State**: Zustand

---

## 📂 Structure

- `frontend/`: The main web application.
- `smart-contract/`: Solidity contracts for immutable records and voting.

---

## 🧪 Quick Start

1. `cd frontend`
2. `npm install`
3. Configure `.env.local`
4. `npm run dev`
